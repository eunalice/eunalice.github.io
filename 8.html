◆ 호위와 아가씨 코스튬

 

<호위와 아가씨 코스튬>이  패키지-이달의 패키지탭 에서 판매됩니다.

 

-패키지 가격이 8900캐시로 할인 되어 판매됩니다.

- 패키지 구성

  : 방어구 4종 ( 투구, 한벌 옷, 주무기, 보조무기 ) + 영웅 칭호

 - 패키지 할인판매 기간

 : 10월 12일 점검 후 ~ 다음 업데이트 전까지

- 단품 판매 여부

 : 단품 판매 진행 안함

-호위와 아가씨 코스튬 간에 세트 효과가 존재합니다.

 
호위와 아가씨 패키지
 
- 획득 가능 칭호 : My Lady
 
 - 획득 가능 칭호 : Sir
 
 - 획득 가능 칭호 : Your Highness   
 
◆ 코스튬 패키지 정가 판매 전환
 

< 이국 여행 패키지 > 가 정가 판매로 전환됩니다.

 

- 무기 미포함 패키지 구성으로, 26400캐시에 판매됩니다.

- 패키지 구성

    : 방어구 5종 (투구, 상의, 하의, 장갑, 신발) + 영웅 칭호

-캐시샵의 패키지 – 코스튬 패키지 탭에서 구매 가능합니다.

- 단품 판매 여부

 : 단품 미 판매

개선사항

-  밀라라우렐 마을에서 자동차 타입의 탈 것 탑승 시 특정 지점에서 탈 것이 기우는 현상이 개선 되었습니다.
- 아처 클래스로 일부 무기 코스튬 장착 시 위치가 기존 코스튬과 다르던 현상이 개선 되었습니다.
- 아처 클래스로 2016 무기 코스튬 장착 시 이펙트 위치가 다소 떨어져 노출되던 현상이 개선 되었습니다.
- 투명한 특수 방어구 장착 후 투명한 부분이 있는 코스튬 장착 시 그래픽이 어색하게 노출되던 현상이 개선 되었습니다.
- 데몰리션 DBF(시크릿 홀로그램 에디션)의 앞 유리 부분이 어색하게 노출되던 현상이 개선 되었습니다.
- 마키나 클래스로 이국 여행 코스튬 장착 시 목,허리,손 부분이 어색하게 노출되던 현상이 개선 되었습니다.
- 아카데믹 클래스로 이국 여행 코스튬 장착 시 목 부분이 어색하게 노출되던 현상이 개선 되었습니다.
- 렌시아 클래스로 이국 여행 코스튬 장착 시 다리 부분이 어색하게 노출되던 현상이 개선되었습니다.
- 별빛의 눈꽃 사슴 탈 것의 발자국이 일부 지역에서 깜빡이는 현상이 개선 되었습니다.
