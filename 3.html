◆ 편의성 개선

 

 길드 미션 납품 편의성 개선 

 

- 길드 미션 납품 시 원하는 납품 횟수를 직접 입력할 수 있도록 개선되었습니다.

- 납품 횟수 입력 칸을 선택하여 직접 횟수를 입력하거나 최대 버튼을 연속으로 입력하여 납품 할 횟수를 자유롭게 변경할 수 있습니다.
- 길드 미션은 한 번에 최대 999회까지 납품이 가능합니다.

 

메인 퀘스트 대상 맵 자동 선택 개선

 

- 진행중인 메인 퀘스트가 있을 경우 던전 선택 화면에서 대상 던전을 자동으로 선택하도록 개선되었습니다.

 

◆ 간편 알림 편의성 개선

 

- 간편 알림에 보유중인 자동 버프형 아이템들의 남은 시간을 표시하도록 개선하였습니다.
- 간편 알림에 자동 버프형 아이템을 최대 5개까지 표시하며 같은 효과를 가진 자동 버프형 아이템 중 남은
  시간이 가장 긴 아이템을 표시합니다.
