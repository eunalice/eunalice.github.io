드네 패스

 

◆ 일일/주간 미션 완료 하고 드네 PASS!

99레벨 달성 시 우측 하단에 드네 패스 아이콘이 출력됩니다. 

드네 패스 아이콘 혹은 F12 클릭 시 크덕이 영상이 출력되며 드네 패스가 시작됩니다~!

 

드네 패스 화면 안내

 

1.드네 패스 시즌 / 패스 적용 여부 / 찜콩 도장 보유 수량이 노출됩니다.
2.드네 패스 진행기간 / 최종 보상까지 필요한 점수 및 보유 점수가 노출 됩니다.
3.드네 패스 일반 보상입니다. ( 모든 용자님들이 임무를 통해 획득 할 수 있습니다. )
4.드네 패스 추가 보상입니다. ( 각 시즌 패스권을 구매한 용자님들이 추가로 획득 할 수 있는 보상입니다. )
5.드네 패스 점수를 획득 할 수 있는 임무로 이동 가능합니다.
6.상단 보상을 얻기 위한 점수 입니다. 해당 점수에 도달하면 보상 획득이 가능하며 버튼 클릭 시 찜콩 도장으로 오픈도 가능합니다.
( 주의 : 찜콩 도장 사용 시 이미 획득한 점수와 상관없이 해당 점수 만큼의 수량이 모두 필요하며, 구매 후 패스 총점도 달성한 점수로 변경 됩니다. )
7.캐시 구입 버튼을 통해 시즌 패스권 및 찜콩 도장 모음을 구입할 수 있습니다.
8.달성 보상 수량이 여러 개일 경우 모두 받기를 사용할 수 있습니다.
     ( 주의 :  모두 받기 버튼 클릭 시 여러 개의 보상이 개별이 아닌 10개씩 함께 수령 됩니다. )
 
 

드네 패스 임무 진행

 

- 임무 리스트에서 패스 일일 임무 / 주간 임무를 진행 합니다.

- 획득한 패스 포인트에 맞게 드네 패스 화면의 점수가 상승합니다.

 

 

찜콩 도장 사용

 

- 임무 달성이 아닌 찜콩 도장을 구입해서 상자 오픈이 가능합니다~!

- 하단의 점수를 클릭하면 찜콩 도장 사용이 가능하며, 크덕이가 해당 단계로 이동합니다.

 

 

드네 패스 보상

 

- 드네 패스 임무 달성 시 획득한 점수로 일반 보상을 획득 할 수 있으며, 드네 패스 구입 시 추가 보상도 함께 지급 됩니다.
